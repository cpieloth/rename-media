# README

Bulk rename of image files based on EXIF date and video files based on encoded date.

```bash
$ rename-media image --directory /tmp/example --prefix IMG_ --suffix _example
✅ Renamed: "/tmp/example/IMG_0002.JPEG" to "/tmp/example/IMG_20251109T163911_example.jpg"
✅ Renamed: "/tmp/example/IMG_0003.jpg" to "/tmp/example/IMG_20251109T163922_example.jpg"
✅ Renamed: "/tmp/example/IMG_0001.JPG" to "/tmp/example/IMG_20251109T163856_example.jpg"
```
